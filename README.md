#  AssetFlow | Enterprise Industrial ERP & Conflict Telemetry Engine

AssetFlow is a high-performance, real-time industrial asset management application designed to eliminate operational bottlenecks, log logistics hardware, and resolve scheduling collisions. Engineered with a low-latency, modern dark-mode user interface and driven by a modular backend routing architecture built for enterprise stability.

---

##  System Architecture & Interface Modules

The frontend layer is architected using server-side compiled Embedded JavaScript (`.ejs`) templates mapped to semantic application endpoints:

* ** Authentication Portal (`/`)**
    * Secure entry checkpoint utilizing credential verification states.
* ** Operational Dashboard (`/terminal`)**
    * Real-time system health telemetry monitoring core industrial outputs.
* ** Asset Directory (`/directory`)**
    * Centralized hardware ledger tracking inventory deployments, valuation thresholds, and location states.
* ** Resource Allocations (`/resources-1`)**
    * Operational shift manager tracking running inventory distribution metrics and active hardware transfers.
* ** Resource Booking (`/resources-2`)**
    * Interactive scheduling ledger optimized for reserving high-demand logistical machinery.
* ** Process Visualization (`/process`)**
    * Node-based infrastructural pipeline visualization displaying hardware dependency trees.

---

##  The Technology Stack

### Frontend Engineering
* **Template Rendering Engine:** Embedded JavaScript (`EJS`) providing dynamic server-side content generation.
* **Style Layer Framework:** Tailwind CSS framework utilizing utilities for consistent layouts.
* **Icon Vector Engine:** Iconify Core architecture rendering rapid, high-performance iconography across navigation headers.

### Backend Infrastructure
* **Runtime Environment:** Node.js server engine.
* **Application Server Framework:** Express.js routing engine mapping application endpoints cleanly.
* **Database Management System:** MongoDB managed via the Mongoose Object Data Modeling (ODM) framework.
